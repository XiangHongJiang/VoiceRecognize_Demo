# VoiceRecognize_Demo

#### 项目介绍
基于科大讯飞的语音识别封装
1：在线语音听写（语音输入）
2：离线语音识别（语音输入：待开发）
3：语音合成（音频转文字）

#### 使用说明

1. 导入讯飞framework，添加依赖库

2. AppDelegate 启动 配置 讯飞识别Id

3. Info.plist添加麦克风权限

4. 开启网络

5. 使用类
